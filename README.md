# 회식 미니게임 (QR 접속)

QR 하나로 전원이 접속해서 같이 즐길 수 있는 **정적 웹페이지**입니다.

## 포함된 게임

- 밸런스 게임: A/B 선택하고 이유 말하기
- 미션 뽑기: 가벼운 미션 랜덤 뽑기
- 칭찬 빙고: 5×5 빙고판 체크
- 룰렛: 항목 편집 가능한 룰렛

## 사용 방법(회식장에서)

1. GitHub Pages(또는 아무 정적 호스팅)에 올립니다.
2. 배포된 URL로 QR을 만들거나, 페이지 상단의 **QR**을 그대로 보여줍니다.
3. 방장 1명이 **“새 시드”**를 눌러 링크를 고정 → 모두 같은 순서로 진행

## 개발/로컬 실행

정적 파일이므로 아무 정적 서버로 열면 됩니다.

```bash
python -m http.server 5173
```

브라우저에서 `http://localhost:5173` 접속.
